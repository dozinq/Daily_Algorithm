> - Binary Search - BOJ2805
>
>   - O(logN)
>
>     ```python
>     N, M = map(int, input().split())
>     data = list(map(int, input().split()))
>     start, end = 1, sum(data)
>     
>     while start <= end:
>         mid = (start + end) // 2
>         cnt = 0
>         for i in data:
>             if i > mid:
>                 cnt += i - mid
>         if cnt >= M:
>             start = mid + 1
>         else:
>             end = mid - 1
>     
>     print(end)
>     ```
